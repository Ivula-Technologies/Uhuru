# Development roadmap

Phase 1 delivers a polished Prologue and Chapter 1 for Windows, Linux, and Web.

1. Foundation — menu, settings, movement, save data, dialogue prototype.
2. Core systems — quests, inventory, journal, NPC schedules, day/night cycle.
3. Vertical slice — first village, five NPCs, intro quests, opening cutscene.
4. Prologue polish — accessibility, audio, educational review, testing.
5. Chapter 1 — the arrival of Europeans, one region and community story at a time.
