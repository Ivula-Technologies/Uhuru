# Visual asset reference boards

The PNGs in this folder are the supplied concept boards for *Uhuru: Eyes of the Land*. Cropped panels in this directory support the title screen presentation; the original boards remain in `assets/source_boards/`.

They are deliberately labelled as reference/concept art. The boards contain captions, layout panels and checkerboard backgrounds, so they should not be mistaken for individually licensed, transparent production sprites. Before a commercial release, replace or recreate every gameplay tile, portrait and UI element as an individually named source asset with documented rights.
